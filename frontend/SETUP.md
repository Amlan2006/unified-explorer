# Setup Instructions

## Installation

```bash
cd unified-explorer/frontend
npm install
```

## Development

```bash
npm run dev
```

Open http://localhost:3000

## Build for Production

```bash
npm run build
npm run preview
```

## Features

- Search across 9 EVM chains including Roburna Testnet
- Smart detection of address/symbol/name
- Beautiful gradient UI
- Real-time results

## Supported Chains

1. Ethereum
2. BNB Smart Chain
3. Polygon
4. Arbitrum One
5. Optimism
6. Avalanche C-Chain
7. Fantom
8. Base
9. Roburna Testnet (Chain ID: 159)
